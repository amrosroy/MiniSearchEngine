# Mini Search Engine in C++

## Features
- File indexing
- Word search
- Frequency-based ranking
- Case-insensitive search

## Data Structures Used
- unordered_map
- vector
- hash-based inverted index

## How It Works
The project builds an inverted index:
word -> filename -> frequency

## Tech Stack
- C++
- STL
